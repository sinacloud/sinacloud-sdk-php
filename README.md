# 新浪云 PHP SDK

## 安装方式

将以下内容加入你的composer.json里。

```json
{
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/sinacloud/sinacloud-sdk-php.git"
        }
    ],
    "require": {
        "sinacloud/sinacloud-sdk-php": "dev-master"
    }
}
```

然后执行 *composer install* 即可完成 *sinacloud-sdk-php* 的安装。
