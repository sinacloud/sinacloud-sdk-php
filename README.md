# SAE PHP SDK

## 安装方式

将以下内容加入你的composer.json里。

```json
{
    "repositories": [
        {
            "type": "vcs",
            "url": "git@git.sae.sina.com.cn:sae/sae-php-sdk.git"
        }
    ],
    "require": {
        "sinacloud/sae-php-sdk": "dev-master"
    }
}
```

然后执行 *composer install* 即可完成 *sae-php-sdk* 的安装。
